# Excel Learning Journey

This repository documents my journey of learning Excel for Data Analyst roles.  
It includes key concepts, practical examples, and mini projects.

---

## 📌 Topics Covered

### 1. Basic Formulas
- SUM, AVERAGE, COUNT, MIN, MAX  
- Simple arithmetic calculations  

### 2. Advanced Formulas
- VLOOKUP, HLOOKUP  
- INDEX-MATCH, IF statements  

### 3. Pivot Tables & Pivot Charts
- Summarize large datasets  
- Create interactive charts from pivot tables  

### 4. Conditional Formatting
- Highlight cells based on rules  
- Color scales, data bars, icons  

### 5. Data Validation & Drop-downs
- Create lists and restrict input  
- Prevent invalid entries  

### 6. Macros & VBA (Beginner-friendly)
- Record macros to automate repetitive tasks  
- Basic VBA scripts  

---

## Goal
To master Excel for data cleaning, analysis, and reporting for real-world datasets.
